Xj3D Browser Command Line Options

Usage: Xj3DBrowser [options] [filename]
      -help                   Prints out this help message
      -fullscreen n           Runs the browser in fullscreen exclusive mode on screen n.  n is optional
      -stereo quad|alternate  Enables stereo projection output
      -antialias n            Use n number of multisamples to antialias scene
      -useMipMaps TRUE|FALSE  Forces mipmap usage on all textures
      -anisotropicDegree n    Forces anisotripic filtering of nTH degree
      -nice                   Do not use all the CPU for rendering
      -captureViewpoints      Generate a screenshot of each viewpoint of the file being loaded
      -zbuffer n              Select how many bits of zbuffer, 8, 16, 24, or 32
      -screenSize w h         Specify the screen size to use
      -disableAudio           Disable the audio output. Useful if you have sound card issues
      -enableOriginManager    Enable the dynamic origin manager in geospatial nodes
      -numLoaderThreads n     Number of threads to use for loading content
      -newt                   Use a GLWindow (NEWT) component to render
      -swing-lightweight      Use a GLJPanel (Swing) component to render
      -swing                  Use a GLCanvas (AWT) component to render
      -title                  Set a title to show in the broswer window frame
