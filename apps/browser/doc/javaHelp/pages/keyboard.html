Xj3D Key Bindings

Viewpoint Navigation

Page-DownNext viewpoint
Page-UpPrevious viewpoint
CTRL-RReset the current viewpoint

Navigation Changes

CTRL-FChange to fly mode. Only works if content has FLY or ANY in Navigation.mode
CTRL-WChange to walk mode. Only works if content has WALK or ANY in Navigation.mode
CTRL-LChange to lookat mode. Only works if content has LOOKAT or ANY in Navigation.mode
CTRL-EChange to examine mode. Only works if content has EXAMINE or ANY in Navigation.mode

Spherical Mode

ALT-SCycle Spherical mode. Only available on Windows.
CTRL-11 Channel rendering
CTRL-22 Channel rendering
CTRL-33 Channel rendering
CTRL-44 Channel rendering

Other

ALT-ACycle Anti-aliasing mode. Increases anti-aliasing till max, then resets
CTRL-NCycle nice mode. Nice mode limits framerate to 50 fps
CTRL-OOpen a file
CTRL-PPrint the user position to the console
ALT-VView the scenegraph. Debug option to print the current scenegraph
ALT-WToggle wireframe rendering style mode
ALT-PToggle point rendering style mode

NOTE: You must click in the Xj3D window for the key bindings to work.
